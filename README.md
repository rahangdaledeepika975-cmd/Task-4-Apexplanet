# Task-4: Personal Portfolio Website

# Internship Program
This project is developed as part of the Web Development Internship at ApexPlanet Software Pvt. Ltd.

# Project Overview
Task-4 focuses on building a complete and functional portfolio website by combining HTML, CSS, and JavaScript.

This project demonstrates structured layout design, responsive styling, form validation, and localStorage integration.

# Features

 1️⃣ Multi-Section Portfolio Website
- Home Section
- About Section
- Projects Section
- Contact Section
- Navigation Bar with smooth scrolling

 2️⃣ Responsive Design
- Mobile-friendly layout
- Adaptive sections for different screen sizes
- Clean and modern UI

 3️⃣ Contact Form with Validation
- Prevents empty submission
- Validates email format
- Shows success messages

 4️⃣ LocalStorage Integration
- Saves user name and email
- Data remains stored after page refresh
- Demonstrates browser storage usage

# Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage API

# How It Works
1. User navigates through different sections using the navbar.
2. User fills out the contact form.
3. JavaScript validates the inputs.
4. Valid data is saved using localStorage.
5. Saved data loads automatically on page refresh.
